# hajiki

見下ろし視点の「おはじき」をイメージしたサンプルです。
画面上に配置された円形を始点にドラッグすることで、ドラッグ方向とは反対に円を飛ばすことができます。

## 実行方法

以下のコマンドで実行できます。

```
npm install
npm run build
npm start
```
